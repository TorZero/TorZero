## Current Engineering Interests

### High-Performance File Transfer

Designing a modern FTP replacement built for real-world networks with latency, jitter, and packet loss. The direction is a UDP-based reliable transport with selective ACKs, congestion control, integrity validation, resumable sessions, encryption, and clear operator diagnostics.

### PyTune Audio Player

Developing a professional Linux desktop audio player using Python, Qt, MPV, FFmpeg, and native FFT modules. The goal is a real audio-driven visual engine, not fake animation: accurate spectrum meters, fast LED/VU response, URL streaming support, and native C++ acceleration for heavy rendering paths.

### Industrial Dashboards

Building professional control-style dashboards for monitoring energy, breakers, telemetry, logs, and device state. Emphasis is on live visibility, fault tolerance, clear state semantics, and commercial-grade operator UI.

### Secure Containers

Researching portable encrypted container workflows using Linux-native tools such as LUKS, loop devices, mount orchestration, web consoles, and portable unlock packages.

### Network Diagnostics

Experimenting with advanced ping/traceroute-style tools that combine ICMP, TCP, UDP, DNS, TLS, and HTTP timing into one operator-friendly diagnostic engine.

## What I Like to Build

I prefer tools that are:

- **Operationally useful**
- **Fast and measurable**
- **Resilient under failure**
